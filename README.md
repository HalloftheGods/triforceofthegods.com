# Triforce of the Gods

**The Codex | Geometric Visualization**

*Part of the Hall of the Gods, Inc. ecosystem.*

## Overview
"Triforce of the Gods" is an interactive 3D geometric visualization and narrative codex exploring the math and philosophy of the spatial web. It features a dynamic fractal Sierpinski Tetrahedron rendered in WebGL using Three.js, paired with a reactive Vue.js user interface.

## Core Features
- **Fractal Geometry**: Dynamic calculation and rendering of recursive tetrahedral branches.
- **Origami Hinge Morphing**: Smooth vertex transitions that programmatically fold a 2D Triforce into a 3D tetrahedron structure.
- **Quantum 4D Projections**: Visualizing tesseract-style continuous 4D rotations using advanced math projections.
- **Narrative Codex**: A synchronized navigation and pagination system linking philosophical axioms to distinct geometric states and 3D animations.

## Tech Stack
- **Three.js** (WebGL 3D Rendering Engine)
- **Vue.js 3** (Reactive UI & State Management)
- **Tailwind CSS v4** (Utility-first styling)
- **Vite** (Next-generation build tool)
- **TypeScript** (Type-safe logic)
